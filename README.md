# Customer Segmentation using K-Means

## 📌 Project Overview
This project applies K-means clustering to group retail customers based on their annual income and spending score.

## 📊 Dataset
Mall Customer Dataset containing:
- Age
- Annual Income
- Spending Score

## ⚙️ Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## 📈 Method
- Used Elbow Method to find optimal clusters
- Applied K-means clustering
- Visualized customer segments

pip install -r requirements.txt
python kmeans.py
